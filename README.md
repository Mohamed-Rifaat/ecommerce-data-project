🛒 E-commerce Data Engineering Project

📌 Overview

This project demonstrates a complete end-to-end data pipeline using Python, SQL Server, and Power BI.

🛠️ Tools & Technologies

- Python (ETL Process)
- SQL Server (Database & Data Modeling)
- Power BI (Data Visualization)

⚙️ Project Steps

1. Generated fake data using Python
2. Designed relational database (Users, Products, Orders, Order Items)
3. Loaded data into SQL Server
4. Performed data analysis using SQL queries
5. Built an interactive dashboard in Power BI

📊 Dashboard Insights

- Total Revenue
- Top Selling Products
- Sales Over Time
- Top Customers

🚀 How to Run
